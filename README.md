# accounting-ledger

## Description
This accounting ledger is a CLI application that can track all financial transactions. This is my capstone project for my Learn to Code Academy.

## Technologies Used
- IntelliJ
- Java
- ArrayList
- LocalDate / LocalTime
- File I/O (BufferedReader)

## Installation
- Clone the repository
- Open in your IDE (e.g., IntelliJ)
- Run the Main class
